⚖️ Suspicion Scoring Engine

Зан үйл бүрийг нэгтгэн magadlal тооцох

score(t) =
  Σ ( w_i × decay(Δt) × conf_i )
  i ∈ all actions

decay(Δt) =
  exp( -Δt / τ )   τ = 600s

risk_level =
  GREEN  if score < 0.40
  YELLOW if score < 0.70
  ORANGE if score < 0.85
  RED    if score ≥ 0.85

🏢 Multi-Tenancy Data Isolation

Tenant бүрийн data ялгаатай namespace-д хадгалагдана

// PostgreSQL — Row Level Security
SELECT * FROM events WHERE tenant_id = current_setting('app.current_tenant_id');

// Redis — Namespaced keys
key = "tenant:abc-123:store:store-42:person:P-247";

// Qdrant — Per-tenant collection
collection = "reid_tenant_abc_123";

// MinIO — Bucket prefix
path = "tenant_abc123/store_42/2026-04-22/event_xyz.mp4";
Isolation guarantee: Tenant А нь tenant Б-ийн data-д хэзээ ч хандаж чадахгүй. Цаашид Enterprise plan-д dedicated GPU + isolated DB schema нэмэгдэнэ (Phase 3).
